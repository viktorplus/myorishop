## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser form → POST /ops | untrusted operator input crosses into the app |
| uvicorn bind address → LAN | anything but loopback exposes an auth-less app |
| user-entered strings → Jinja2 templates | rendered back into HTML |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-1-01 | Tampering | POST /ops form input | mitigate | FastAPI Form typing (`qty_delta: int`) → 422 on garbage; SQLAlchemy ORM parameterized queries only, zero raw f-string SQL (ASVS V5) |
| T-1-02 | Information Disclosure | uvicorn bind | mitigate | `--host 127.0.0.1` hard-coded in run.bat; automated grep gate rejects 0.0.0.0 |

| T-1-08 | Denial of Service | double-submit creating duplicate ops | accept (partial mitigation) | hx-disabled-elt disables the button during flight; UNIQUE(device_id, seq) makes any race loud; single local operator = low risk |
