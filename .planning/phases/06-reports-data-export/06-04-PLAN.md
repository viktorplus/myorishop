    - stream_products_csv/stream_sales_csv/stream_customers_csv each return a FastAPI StreamingResponse whose byte stream, when decoded and reassembled, starts with EXACTLY ONE UTF-8 BOM (b"\xef\xbb\xbf") at the very start — not one per chunk (RESEARCH: encoding every chunk with utf-8-sig would repeat the BOM and corrupt the file).

    - stream_products_csv's body, read back with csv.reader after stripping the BOM and using delimiter=";", round-trips to the same row count and header as the seeded products (header: Код;Название;Категория;Закупка;Продажа;Каталог;Остаток).
    - A product/customer name beginning with "=", "+", "-", or "@" is written with a leading apostrophe prefix (CSV/formula-injection hardening) so Excel never interprets it as a formula on open.
    - No function in this module accepts a filename, path, or any other parameter from outside the module — every export is a full, unfiltered table dump (matches the V12 pattern already established by app/routes/backup.py).
