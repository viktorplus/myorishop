| Boundary | Description |
|----------|-------------|
| GET /sales/customer-search ?q → LIKE | Untrusted query reaches the customer search |
| POST /sales/customer → customers table | Untrusted quick-create name/surname/consultant stored |
| Selected chip / picker rows → templates | Untrusted customer names rendered in the sale form |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-4-02 | Tampering | /sales/customer-search `q` | mitigate | Reuses `search_customers` ORM `.contains(q_lc, autoescape=True)` — parametrized + LIKE escaping |
| T-4-01 | Tampering/Info | customer names in picker + chip | mitigate | Jinja autoescape ON; `<mark>` literal; never `|safe` |
| T-4-05 | Tampering | POST /sales/customer binding | mitigate | Explicit `Form(...)` params only; register_sale reads only the named `customer_id` |
