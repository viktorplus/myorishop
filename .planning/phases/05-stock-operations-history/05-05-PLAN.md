## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser → GET /history | Untrusted type/product/page query params from filters + pagination. |
| stored op fields → /history render | Product names, payload notes, created_by are untrusted-at-rest text rendered in the table. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-18 | Tampering | product name / payload note / created_by in /history | mitigate | Jinja autoescape ON; NEVER `|safe` on any history cell (stored-XSS T-4-01 precedent) — the primary render surface for all operator free-text. |
| T-05-19 | Denial of Service | unbounded ledger load | mitigate | LIMIT/OFFSET pagination (page_size 50 + fetch-one-extra sentinel); the whole ledger is never materialized in one response (D-15). |
| T-05-20 | Tampering | tampered type/product filter params | mitigate | Portable ORM parameterized `.where()` (no raw SQL → no injection); an unknown type value yields the unfiltered/empty view, not an error. |
| T-05-21 | Info disclosure | GET /history exceptions | mitigate | Read-only query; on unexpected failure FastAPI returns a generic error — no operator input is echoed into a stack trace (V7). |
| T-05-SC | Tampering | pip installs | accept | No package installs this phase (internal wiring only) — 05-RESEARCH § Package Legitimacy Audit: N/A. |
</threat_model>
