Output: `lookup_prefill()` returns a combined `catalog` source for unknown codes; `/receipts/lookup` fills name + cost/catalog (never sale) for that source; product-add's existing autofill routes carry an explicit PRICE-02/PRICE-03 reference.
</objective>
